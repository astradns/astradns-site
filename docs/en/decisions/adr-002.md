# ADR-002: Policy Scope in MVP

- Status: Accepted
- Decision: Support namespace scope in MVP and defer workload/domain scope.
- Why: Namespace identity cannot be inferred reliably in the current CoreDNS forwarding path.
- Consequence: `ExternalDNSPolicy` is API-ready, while strict enforcement evolves with the direct datapath roadmap.
