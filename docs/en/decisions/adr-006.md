# ADR-006: Namespace Enrichment

- Status: Accepted
- Decision: Accept namespace="unknown" in MVP and add IP-to-namespace enrichment in post-MVP.
- Why: Source IP alone is insufficient for accurate namespace attribution in the current data path.
- Consequence: The agent will later use pod informer data to enrich logs and metrics.
