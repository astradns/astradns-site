# ADR-007: Query Logging Strategy

- Status: Accepted
- Decision: Emit structured query logs from the agent to stdout with selectable logging modes.
- Why: It integrates with existing customer log pipelines without coupling to one backend.
- Consequence: Operators can tune visibility and volume (`full`, `sampled`, `errors-only`, `off`).
