# ADR-001: Data Path Interception

- Status: Accepted
- Decision: Use the NodeLocal DNS pattern where AstraDNS Agent binds to a link-local address on each node and CoreDNS forwards external queries to it.
- Why: It is a proven architecture with low operational risk and per-node isolation.
- Consequence: Each node gets independent cache and resolver behavior.
