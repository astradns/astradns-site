# ADR-004: Failover Strategy

- Status: Accepted
- Decision: Use engine-native failover for MVP, then add an agent-level circuit breaker in post-MVP phases.
- Why: It reduces implementation risk while preserving observability and recovery planning.
- Consequence: Reliability improves incrementally without delaying MVP delivery.
