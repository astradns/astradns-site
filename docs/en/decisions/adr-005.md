# ADR-005: Naming and Positioning

- Status: Accepted
- Decision: Use the AstraDNS name and the `dns.astradns.com` API group.
- Why: It gives clear identity and Kubernetes-native naming consistency.
- Consequence: Documentation, charts, and CRDs align around one stable naming model.
