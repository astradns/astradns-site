# ADR-003: Modelo de Métricas

- Status: Aceito
- Decisão: Manter métricas do MVP em escopo de nó.
- Motivo: Labels por nó têm cardinalidade controlada; labels por pod/domínio elevam risco operacional.
- Consequência: Métricas por namespace ficam para a fase com datapath direto.
