# ADR-005: Nomenclatura e Posicionamento

- Status: Aceito
- Decisão: Adotar o nome AstraDNS e o grupo de API `dns.astradns.com`.
- Motivo: Reforçar identidade do projeto e consistência com padrões Kubernetes.
- Consequência: CRDs, Helm e documentação convergem para o mesmo modelo de nomenclatura.
