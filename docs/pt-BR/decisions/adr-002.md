# ADR-002: Escopo de Política no MVP

- Status: Aceito
- Decisão: Suportar escopo por namespace no MVP e adiar escopos por workload/domínio.
- Motivo: O caminho atual de encaminhamento via CoreDNS não permite inferência confiável de namespace.
- Consequência: O `ExternalDNSPolicy` permanece pronto na API, enquanto a aplicação evolui com o datapath direto.
