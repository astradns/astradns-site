# ADR-007: Estratégia de Logging de Consultas

- Status: Aceito
- Decisão: Publicar logs estruturados de consulta no agent via stdout com modos configuráveis.
- Motivo: Integrar com pipelines de logs já existentes sem acoplamento a um backend específico.
- Consequência: Operação com controle de volume (`full`, `sampled`, `errors-only`, `off`).
