# ADR-006: Enriquecimento de Namespace

- Status: Aceito
- Decisão: Aceitar namespace="unknown" no MVP e adicionar enriquecimento IP->namespace no pós-MVP.
- Motivo: O IP de origem, isoladamente, não resolve namespace com precisão no caminho atual.
- Consequência: O agent evolui para usar informer de pods e enriquecer métricas/logs.
