# ADR-001: Interceptação do Caminho de Dados

- Status: Aceito
- Decisão: Usar o padrão NodeLocal DNS, com o AstraDNS Agent em endereço link-local por nó e encaminhamento externo pelo CoreDNS.
- Motivo: Arquitetura comprovada, com baixo risco operacional e isolamento por nó.
- Consequência: Cada nó mantém cache e resolvedor próprios.
