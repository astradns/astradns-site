# ADR-004: Estrategia de Failover

- Estado: Aceptado
- Decisión: Usar failover nativo del motor en MVP y agregar circuit breaker en el agent en post-MVP.
- Motivo: Entregar resiliencia incremental sin retrasar la entrega inicial.
- Consecuencia: El roadmap mejora confiabilidad por fases, manteniendo observabilidad.
