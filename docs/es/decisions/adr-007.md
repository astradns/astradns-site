# ADR-007: Estrategia de Registro de Consultas

- Estado: Aceptado
- Decisión: Emitir logs estructurados de consultas desde el agent por stdout con modos configurables.
- Motivo: Integración con pipelines de logs existentes sin acoplarse a un backend específico.
- Consecuencia: Operación con control de volumen (`full`, `sampled`, `errors-only`, `off`).
