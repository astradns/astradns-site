# ADR-003: Modelo de Métricas

- Estado: Aceptado
- Decisión: Mantener métricas de MVP en alcance de nodo.
- Motivo: Las etiquetas por nodo tienen cardinalidad controlada; por pod/dominio elevan el riesgo operacional.
- Consecuencia: Métricas por namespace quedan para la fase con datapath directo.
