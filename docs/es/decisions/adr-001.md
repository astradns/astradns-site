# ADR-001: Intercepción de la Ruta de Datos

- Estado: Aceptado
- Decisión: Usar el patrón NodeLocal DNS, con AstraDNS Agent en dirección link-local por nodo y reenvío externo desde CoreDNS.
- Motivo: Arquitectura probada, con bajo riesgo operacional y aislamiento por nodo.
- Consecuencia: Cada nodo conserva caché y resolvedor independientes.
