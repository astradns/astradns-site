# ADR-005: Nombre y Posicionamiento

- Estado: Aceptado
- Decisión: Adoptar el nombre AstraDNS y el grupo de API `dns.astradns.com`.
- Motivo: Consolidar identidad del proyecto y consistencia Kubernetes.
- Consecuencia: CRDs, Helm y documentación se alinean en un mismo modelo de nomenclatura.
