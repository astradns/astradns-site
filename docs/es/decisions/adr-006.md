# ADR-006: Enriquecimiento de Namespace

- Estado: Aceptado
- Decisión: Aceptar namespace="unknown" en MVP y agregar enriquecimiento IP->namespace en post-MVP.
- Motivo: La IP de origen por sí sola no permite resolver namespace con precisión en el camino actual.
- Consecuencia: El agent evolucionará con informer de pods para enriquecer métricas y logs.
