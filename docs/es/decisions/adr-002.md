# ADR-002: Alcance de Políticas en MVP

- Estado: Aceptado
- Decisión: Soportar alcance por namespace en MVP y diferir alcance por workload/dominio.
- Motivo: El datapath actual por CoreDNS no permite inferencia confiable de namespace.
- Consecuencia: `ExternalDNSPolicy` permanece listo a nivel de API mientras la imposición evoluciona con datapath directo.
